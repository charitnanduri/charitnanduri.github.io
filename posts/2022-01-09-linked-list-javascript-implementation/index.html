So, what is a Linked List?#

A Linked List is a linear collection of nodes in memory. Each node has a pointer referencing another node. Let me illustrate and describe the Linked List in detail.

The first node which is also called the head represented above contains two things:

  1. Data which contains a value of 1000.
  2. A Pointer (also called a reference) contains the address of the second node and is represented as NEXT.

Every Linked List starts with a HEAD and ends with a TAIL. If the HEAD is NULL then we can say that the Linked List is empty. A TAIL has a pointer value of NULL (indicating that it is the last item in the collection).

Often Linked Lists are compared to arrays, this is because we can store data in arrays in a linear form similar to Linked Lists. The biggest difference is that Linked Lists are not stored in the contiguous form in memory but rather connected to each other by reference.

Key things to know about Linked Lists#

  • Memory allocation: Linked Lists are stored randomly in memory in a non-contiguous form so allocating resources need not be pre-defined.
  • Dynamic size: The size of the Linked List is not static which means they can shrink and grow during execution.
  • Performance: Inserting and deleting is fast and easy but traversing is slow because of their linear structure
  • Memory efficiency: A Linked List has an effective way of utilising memory as it can track chunks of memory that is available for allocation.
  • Foundation for other structures: Many important data structures such as stacks, queues, hash tables and graphs are implemented using a Linked List.

insertAtTail(data) {
    const newTail = new Node(data);
    // Check: if list is empty then last node will be head
    if (!this.head) {
      this.head = newTail;
    } else {
      // if not empty list, traverse to tail node
      let tail = this.head;
      while (tail.next) {
        tail = tail.next;
      }
      tail.next = newTail;
    }
    this.sizeOfLinkedList++;
  }

getByIndex(index) {
    // check to see if the index is valid or not
    if (index < 0 || index >= this.length) {
      return null;
    }
    // looping to find the current value using the pointer next
    let current = this.head;
    for (let i = 0; i < index; i++) {
      current = current.next;
    }
    return current;
  }

insertAtIndex(index, data) {
    //check to see if index is head
    if (index === 0) {
      return this.insertAtHead(data);
    }
    //Assign (index-1) value to previous variable
    const previous = this.getByIndex(index - 1);
    if (previous == null) {
      return null;
    }
    previous.next = new Node(data, previous.next);
    this.sizeOfLinkedList++;
  }

Summary#

Linked Lists are basic yet powerful data structures that can help us define structure complex data. There are two more types of Linked Lists known as Doubly-Linked List and Circularly Linked List.

Further reading into Linked Lists includes time complexity analysis using Big O notation.
